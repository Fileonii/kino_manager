# Aplikacja obsługująca Kino

Aplikacja wykorzystująca technologię React+Redux, wspomagająca pracę kina, poprzez wybór miejsc na sali kinowej.

## Instalacja

Aby zainstalować należy wpisać

```bash
cd client
npm i
npm start
```
oraz
```bash
cd server
npm i
npm run api
```
